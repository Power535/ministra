Memory

Žaidimo pradžioje žaidėjui parodoma 10 kortelių porų. Po to kortelės apverčiamos. Naudodamas navigavimo mygtukus, žaidėjas išsirenka kortelę ir, paspaudęs OK mygtuką, ją atverčia. Po to tarp užverstų kortelių reikia prisiminti ir surasti kortelę, identišką atverstajai. Jei žaidėjas teisingai surado „porą“, abi kortelės lieka atverstos. Priešingu atveju atversta kortelė vėl užverčiama. Užduotis – mažiausiu ėjimų kiekiu atversti visas kortelių poras. Žaidimas Memory lavina atmintį ir erdvinį mąstymą.
