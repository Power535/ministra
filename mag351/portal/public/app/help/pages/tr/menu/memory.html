Memory

Oyun başlangıcında oyuncuya 2 çift kâğıt verilir. Bundan sonra kâğıtlar ters çevrilir. Oyuncu navigasyon tuşarı ile kâğıdı seçip OK tuşu ile onu açtırıyor. Bundan sonra kapalı olan kâğıtlardan açıldığı kâğıda eş olanı bulmak lâzım. Eşi bulması halinde ikisi de açık oluyor. Aksî halde ikisi de kapalı tutulur. Oyuncunun ödevi en az sıra sayısı kullanıp tüm kâğıt çiftlerini açtırmaktır. Memory oyunu hafızayı ve uzaysal zekâyı geliştirir.
